## `> overview`

Este projeto foi construído como uma aplicação front-end moderna para representar minha identidade profissional de forma mais visual, técnica e memorável.

A proposta foi ir além de um portfólio estático, desenvolvendo uma interface com:

- visual tecnológico
- navegação fluida
- responsividade manual
- microinterações
- animações customizadas
- organização modular de arquivos

O resultado é uma experiência voltada tanto para **recrutadores** quanto para **desenvolvedores**, combinando apresentação profissional com execução técnica bem estruturada.

## `> project_vision`

Este portfólio foi pensado como uma interface com foco em:

- identidade visual tecnológica
- performance visual
- clareza na apresentação dos projetos
- separação bem definida de responsabilidades
- experiência mais imersiva para o usuário

O objetivo foi criar uma aplicação que demonstrasse não apenas conteúdo, mas também **cuidado com arquitetura, design e experiência de uso**.

## `> technical_highlights`

### Principais destaques da implementação

- layout responsivo com adaptação manual
- background animado com **Canvas API**
- efeito de digitação personalizado
- organização modular de CSS e JavaScript
- renderização dinâmica de conteúdo
- sistema de filtros para projetos
- modal dinâmico para exibição detalhada
- separação entre lógica, estilo e estrutura
- foco em legibilidade e experiência visual

## `> engineering_decisions`

### Estrutura adotada

- **HTML semântico** para melhor organização estrutural
- **CSS modularizado** para separar responsabilidades visuais
- **JavaScript ES6** para interatividade e manipulação da interface
- **Canvas API** para reforçar identidade visual com background animado
- **separação Desktop / Mobile** para maior controle do comportamento responsivo

### Decisões técnicas

- sem frameworks, para manter controle total da estrutura
- responsividade manual, priorizando precisão visual
- animações leves, preservando performance
- base preparada para futuras evoluções ou migração para framework
